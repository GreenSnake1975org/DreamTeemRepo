  1. git init
  2. git status
  3. git add #file | *
  4. git add *.* : add to stage all FILES in current FOLDER
  5. git add . : add to stage all FILES in current FOLDER AND SUBFOLDERS
  6. git commit -m "#message"
  7. git log --graph
  8. git diff HEAD : delta between HEAD and current unCommited stages
  9. git diff HEAD~10 : delta between current state of branch (unCommited) and HEAD 10 steps backward
  10. git diff HEAD~10 HEAD : delta between current HEAD and HEAD 10 steps backward
  11. git diff #SHA1_for_commit01 #SHA1_for_commit10 : delta between stage on commit with SHA1_01 and stage on commit with SHA1_10
  12. git reset --hard HEAD~3 : UNDO (erase) 3 LAST commit's
  13. git reset --hard HEAD~ : UNDO (erase) 1 LAST commit
  14. git reset --soft HEAD~3 : go back on 3 steps and make 3 last stages uncommited
